# Diagnostische Adaptieve Toets Wiskunde

Begin 4de jaar secundair onderwijs (ASO) — Wat weet je nog?

**Inhoud:** Rekenregels, rekenvaardigheden en formules uit jaar 1–3  
**Type:** Adaptief (per competentiegebied, 0–3 niveaus)  
**Tijdsduur:** 2 × 45 minuten  
**Domein:** classroomatheneum.be

## Technisch

- React + Vite + TypeScript + Tailwind CSS
- KaTeX voor wiskunde-rendering
- Supabase (PostgreSQL + Auth)
- Google OAuth via @classroomatheneum.be
- Hosting: Vercel (auto-deploy vanaf main)

## 7 Competentiegebieden

| Deel 1 (±45 min) | Deel 2 (±45 min) |
|---|---|
| A - Getallenkennis & verzamelingen | E - Functies |
| B - Bewerkingen & rekenregels | F - Meetkundige formules |
| C - Algebraïsche uitdrukkingen & formules | G - Goniometrie & Pythagoras |
| D - Vergelijkingen & ongelijkheden | |
